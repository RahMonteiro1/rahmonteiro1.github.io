# RoughFire

Portfólio de projetos, experimentos e aprendizados de alguém que acredita que toda habilidade começa com uma pergunta.

🔗 [rahmonteiro1.github.io](https://rahmonteiro1.github.io)

## Sobre

Este site documenta minha jornada de aprendizado em desenvolvimento de software, análise de dados e eletrônica, incluindo as tentativas, os impasses e as
conclusões extraídas de cada experimento.

## Construído com

- [Jekyll](https://jekyllrb.com/)
- Tema base: [Centrarium](https://github.com/bencentra/centrarium) por Ben Centra (MIT), customizado com identidade visual e estrutura de conteúdo próprias
- Hospedado via GitHub Pages, com deploy automatizado por GitHub Actions
